# Drone
I made a drone featuring , made with SoC board  based on an ESP-class Processor,with an OV2640 Camera.

## Features

- Battery input with switch
- Onboard power regulation
- Camera interface via 24pin OV2640 FPC
- USB OTG support
- SD/MMC storage
- Gyroscope sensor interface
- MOSFET-controlled outputs
- Buzzer output

  ## Hardware blocks

- Battery Input + Switch
- Power Regulator
- SoC
- OV2640 FPC 24P
- USB OTG
- SD/MMC
- Gyro
- MOSFETs
- Buzzer / charge check
- Camera supply rails
